# `GamePadControl`

Simulate pressing or releasing (or both) gamepad buttons.

## Example Use
`GamePadControl.None`

## Possible Values
* `None`
* `DPadUp`
* `DPadDown`
* `DPadLeft`
* `DPadRight`
* `Start`
* `Back`
* `LeftStickClick`
* `RightStickClick`
* `LeftShoulder`
* `RightShoulder`
* `Guide`
* `A`
* `B`
* `X`
* `Y`
* `LeftTrigger`
* `RightTrigger`
* `LeftStickLeft`
* `LeftStickRight`
* `LeftStickUp`
* `LeftStickDown`
* `RightStickLeft`
* `RightStickRight`
* `RightStickUp`
* `RightStickDown`
* `LeftStickAsAnalog`
* `RightStickAsAnalog`
* `DPadAsAnalog`
